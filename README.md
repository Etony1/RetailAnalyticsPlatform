# Retail Analytics Platform

An end-to-end retail analytics platform designed to simulate a real-world retail business. The project demonstrates operational database design, transactional data modeling, business rule implementation, ETL pipelines, Databricks Medallion Architecture, SQL analytics, and Power BI dashboards.

## Goals

- Design a normalized operational database
- Simulate realistic retail business operations
- Implement production-style ETL pipelines
- Build a Medallion Architecture in Databricks
- Create an analytical star schema
- Deliver executive dashboards in Power BI
