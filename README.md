# Overview

This project demonstrates the complete lifecycle of a retail analytics platform—from synthetic transactional data generation through enterprise-grade data validation, ETL processing, Medallion Architecture, analytical modeling, and executive dashboards.

The goal is to replicate the architecture and workflows used by modern data engineering teams while showcasing best practices in Python, SQL, PySpark, Databricks, and Power BI.
